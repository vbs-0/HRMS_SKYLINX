👑 Platform Owner

SUPER_ADMIN / SYSTEM_OWNER. Runs the platform: tenants, plans, billing, global config. Bypasses tenant scoping — cross-tenant oversight. Not day-to-day HR.

🛡️ HR Admin

Full HR operations within one company: employees, payroll, leave, recruitment, settings. The primary administrator.

👔 Manager

Team-scoped. Approves their reports' requests (leave, attendance, expense), views team data. No configuration.

💼 Employee

Self-service. Manages own data, submits requests, views own records (payslip, leave balance, documents).
